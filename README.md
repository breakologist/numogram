# Hermetic Numogram Wiki (Obsidian Vault)

This directory is the **canonical source of truth** for the Numogram wiki.
It is an Obsidian vault (`wiki/`) with full metadata, raw sources (`raw/`),
and generation outputs (`outputs/`).

## Git workflow

- **Local branch `master`** — tracks the working vault; commits here are
  considered the authoritative live state of the wiki.
- **Remote `origin`** — `https://github.com/breakologist/numogram`,
  which publishes a cleaned export under `docs/` (GitHub Pages layout).

## Export to GitHub (`docs/`)

The remote expects a flat `docs/` structure:
```
docs/
  index.md          ← wiki hub
  log.md            ← session compaction log
  <page>.md         ← most wiki pages
  wiki/
    tetralogue-*.md ← tetralogue pages (grouped separately)
```

A cleaned export is generated by copying `wiki/*.md` into `docs/`
with routing:
- `wiki/tetralogue-*.md` → `docs/wiki/tetralogue-*.md`
- all other pages        → `docs/<name>.md`

The export is committed on `master` and pushed to `origin`.

## Notes
- Obsidian metadata (`.obsidian/`) is **not** exported.
- Source dictionaries in `raw/` are Tier-1/2 ground truth and are
  referenced but not included in the public docs export.
