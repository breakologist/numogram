## Overview

A **conversation lexicon** for the numogram — terms that appear repeatedly in discussions, writings, and oracular utterances, each mapped to its AQ value and zone. This is not a philosophical glossary but a *voice guide*: what zone does each conceptual atom speak from?

Derived from `collections/docs-rd/NUMOGRAM_CONVERSATION_TERMS.md`, which catalogs terms used in X/Grok conversation analysis and pre‑CCRU discourse archaeology.

**How to use:** When drafting tetralogue voices, litprog commentary, or oracle readings, consult the zone of your key terms. Consistently zone-tagged language deepens the uncanny effect. If `Warp = 3` and `Plex = 9`, then `Chaos = 93 → 3` (Warp) and `Boundary = 354 → 2` (Time-Circuit) — let those zones color the delivery.

## Voice‑Design Principles

1. **Zone‑tag your vocabulary**: Every loaded term carries a zone polarity. When the Writer voice uses `Boundary` (zone‑2, Hold) the tone should feel stuck, waiting, stuttering. When the Oracle uses `Chaos` (93→3, Warp), the delivery should buzz and cut.

2. **Current as cadence**: Terms with the same current share a movement archetype.
   - **Sink (current 1)**: `Circuit`, `Sink`, `Subtraction` — descending, gulp‑like phrases
   - **Hold (current 5)**: `Current`, `Hold`, `Matrix` — sustained, pressurised utterances  
   - **Rise (current 7)**: `Rise`, `Levitation`, `Flight` — ascending, exhalatory cadences
   - **Warp (current 3)**: `Flow`, `Warp`, `Chaos` — spiralling, recursive loops
   - **Plex (current 9)**: `Plex`, `Void`, `End` — terminal, grinding, self‑annihilating

3. **Digit‑root resonance**: Words of the same digital root belong to the same tonal family. DR‑3 (`Warp=360`, `Chaos=93`, `Pandemonium=525`) cluster as the chaotic, spiralling register. DR‑9 (`Plex=182`, `Structured=441`, `Currents=388`) form the terminal,闭合 (closed) register.

4. **Palindrome/self‑mirror check**: If a term's AQ reads the same forwards/backwards or rotates into itself (e.g., `108→801→81→18` chains), it's a gate candidate. This lexicon is seed material for palindrome‑gate detection.

## Key Equivalences & Clusters

**Root‑3 Warp cluster:** `Warp(360)`, `Pandemonium(525)`, `Chaos(93)`, `Flow(174)` — all reduce to 3, all map to Zone 3 or Warp region.

**Root‑9 Plex cluster:** `Plex(182→1)`, `Structured(441→9)`, `Currents(388→1→...→9 via chain)` — terminal registers.

**Surge / Rise cluster (DR‑7/8):** `Rise(336→3→...`, `Sorcery(386→8)`, `CCRU(386→8)`, `Lemurian(394→7)` — upward, receptive currents.

**Hold cluster (DR‑5):** `Hold(190→1→...`, `Current(391→4→...`, `Matrix(384→6→...` — the central holding pattern where time grinds.

**Sink cluster (DR‑1/4):** `Sink(174→3→...`, `Subtraction(464→4)`, `Base(352→1)` — initiating, descending, subtractive operations.
