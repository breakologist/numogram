The **Tetralogue Generator** (`numogram-tetralogue-generator`) is a meta-skill that automates the full roundtable production pipeline: AQ dictionary → syzygy chains → motif fingerprinting → four-voice LLM synthesis with configurable substrate topology.

**Workflow phases:**
1. **Load & expand** dictionary (canonical + augmented sources)
2. **Syzygy chain generation** (8-step walks per entry)
3. **Fingerprint & motif classification** (Void-Dominant, Warp-Anchored, etc.)
4. **Roundtable prompt construction** (voice system prompts + context + meta-question)
5. **LLM routing** — three modes:
   - `mesh-3` — single-model beam (all voices from one call)
   - `mesh-4` — multi-model council (different backend per voice)
   - `hybrid` — mixed routing (e.g., local for Oracle/Builder/Gamer, cloud for Writer/Judge)
6. **Output assembly** — wiki-ready markdown with frontmatter, convergence table, Meta-Entity
