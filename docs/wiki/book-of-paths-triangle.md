# The Book of Paths Triangle — 84 Levels, Three Voices

Source: Aamodt, *Unleashing the Numogram*, Book of Paths (lines 3143–3820) and Glossary (lines 3821+)

84 paths. Each is a name, a direction, a method, tests, and an outcome. The Horowitz letter says 84 bronze tablets were removed from the Shu excavation. The number is not a coincidence. It is the same text.

## The Structure

Every path follows this template:

```
N. [Name]
[Direction — where the path leads]
The path favours [method combination].
Superior subtlety [action — opens roads / leads nowhere / enters labyrinth].
[Compliance/Resistance/Pure variant/Poised entanglement] prevails.
[1–7 or Twinned] tests on the way.
[Outcome sequence — 1–4 events]
```

**Direction types:** Ascent, Descent, Waiting (in Rising/Falling Drift or Lesser Depths), Sunken Track, Twin Heavens, Crossing between Drifts, Sudden, Endless.

**Methods:** Patience, Subtlety, Activity — alone, twofold, threefold, or "all methods entwined."

**Subtlety actions:** Opens the first/second/three hidden roads. Leads nowhere. Enters the spiral labyrinth.

**Outcomes:** Breakthrough. Fluid evolution. Possession. Captivation. Lucid delirium. Swirling confusion. Ominous transition. Difficulties annihilated. Dubious inheritance. Burning excitement. Arid tension finds release. Strategic withdrawal.

## I. The Builder Designs 84 Levels

Each path is a level. The template is the level's *grammar*. The name is its *title screen*.

### Level Grammar → Level Architecture

| Path Element | Level Element |
|-------------|---------------|
| Name | Level title + aesthetic theme |
| Direction | Spatial orientation — is the level ascending, descending, or lateral? |
| Method | Player strategy emphasis — patience = stealth, subtlety = exploration, activity = combat |
| Subtlety action | Number of hidden rooms/side paths available |
| Compliance/Resistance | Environmental hostility — does the level assist or oppose the player? |
| Tests on the way | Number of mandatory encounters (combat, puzzle, or hazard) |
| Outcome sequence | Level climax type + reward/punishment tier |

### Worked Examples

**Path 1. Original Subtraction**
- Direction: Ultimate descent through the Depths → vertical shaft dungeon, all down
- Method: Repeated patience linked by subtlety → stealth-exploration hybrid
- Subtlety: Opens three hidden roads → three optional rooms
- Compliance prevails → environment is navigable, not hostile
- 3 tests → three mandatory encounters
- Outcomes: Immersive nightmares → ominous transition → difficulties annihilated
- **Level design:** A three-floor descent. Each floor has one mandatory encounter and one optional room. The environment is compliant — no traps, no ambushes. The "immersive nightmares" are atmospheric: the level gets darker, the sounds get stranger, but nothing attacks until the ominous transition on the final floor, where the exit is guarded and the difficulties are *annihilated* — not defeated, erased.

**Path 34. Celestial Capture**
- Direction: Sudden ascent to the Twin Heavens → teleportation, no gradual climb
- Method: Subtlety, until all methods entwines → exploration that suddenly demands everything
- Subtlety: Enters the spiral labyrinth → non-linear, recursive geometry
- Pure resistance → maximum hostility
- 2 tests → only two encounters, but both extreme
- Outcome: Possession by swirling confusion
- **Level design:** The player enters a room and is *launched upward*. No stairs. No ramps. The Twin Heavens appear instantly. The level is a spiral labyrinth — corridors that loop back on themselves. Two encounters: the first tests subtlety (find the path through the spiral), the second tests everything (the labyrinth fights back). The outcome is possession — the level takes the player over. The exit doesn't lead out. The exit leads *deeper in*.

**Path 36. Vortical Coincidence**
- Direction: Endless waiting in the Twin Heavens → no exit, no descent, stasis
- Method: All methods entwined → patience + subtlety + activity simultaneously
- Subtlety: Enters the spiral labyrinth → recursive geometry
- Poised entanglement → neither compliance nor resistance, both at once
- Twinned tests make the way → tests occur in pairs, each pair is one test split in two
- Outcome: Between lucid delirium and swirling confusion
- **Level design:** This is the infinite level. There is no exit. The player waits in the Twin Heavens and the level *continues generating*. Each twinned test is a mirror encounter — the same enemy, the same puzzle, but one version is lucid (clear, solvable) and the other is confused (distorted, paradoxical). The player must solve both simultaneously. The outcome is not completion. The outcome is *being between states* — lucid and confused at the same time. This is the roguelike equivalent of a meditation.

**Path 84. Compressed Termination**
- Direction: Descent → straight down, no elaboration
- Method: Subtlety → pure exploration, no combat emphasis
- Subtlety: Opens the second hidden road → one optional room
- Pure resistance → maximum hostility
- 1 test → one encounter
- Outcome: Ominous transition
- **Level design:** One room. One enemy. One hidden room if you're subtle enough to find it. The level is a compression of everything that came before — 83 levels of increasing complexity distilled into a single encounter. The ominous transition is the game ending. Not with a fanfare. With a compression. The entire Book of Paths collapses into this one room.

### The 84-Level Sequence

The paths are not random. They form a progression:

```
Paths 1–9:    Descent arc (original subtraction → sudden flight)
               The player goes down. The numogram is explored from above.

Paths 10–20:  Transition arc (jagged flight → alien intervention)
               The player crosses between regions. Methods entwine.

Paths 21–35:  Ascent arc (supreme comprehension → erratic flight)
               The player rises. Resistance increases. The Twin Heavens appear.

Path 36:      Vortical Coincidence — THE PIVOT
               Endless waiting. All methods. Poised entanglement.
               This is Zone 6 in the Warp — the centre of the spiral.

Paths 37–50:  Complex arc (indirect escape → various)
               Paths branch. Subtlety enters the labyrinth. Outcomes multiply.

Paths 51–83:  Terminal arc (tidal → buried → plunging)
               The player sinks. The Sunken Track dominates. Ominous transitions accumulate.

Path 84:      Compressed Termination
               One room. One test. The end.
```

Path 36 is the numogram's Zone 6 — the Warp centre. Everything before it descends and ascends around this pivot. Everything after it spirals downward toward the Plex.

## II. The Writer Gives the Paths Their Sound

**The first nine paths sound like breathing slowing down.**

"Original Subtraction." The name itself is a reduction. Not addition, not multiplication — subtraction. The original operation. Taking away. The path descends through the Depths and the writing should feel like sinking into water that is warmer than expected. Not cold. Warm. The compliance means the water *wants* you to descend.

"Extreme Regression." Not just regression — *extreme* regression. Threefold patience. The player waits and waits and waits. The sound of this path is a held breath. The Rising Drift leads to ultimate descent — you think you're going up but the current pulls you down. The writing should make the reader feel the disorientation of ascending while sinking.

"Abysmal Comprehension." The seventh test. Burning excitement provokes breakthrough into immersive nightmares. This is the first path where the writing should *accelerate*. Six paths of patience and then — burning. The breakthrough is not relief. It is the moment the nightmare becomes immersive. You don't wake from the dream. You go deeper in.

It is the ground ceasing to exist.

**Path 34 sounds like a trap snapping shut.**

"Celestial Capture." Not ascent — *capture*. The Twin Heavens don't welcome you. They seize you. The direction is "sudden" — no gradual climb, no preparation. One moment you're in the Depths, the next you're in the Heavens, and the Heavens are a spiral labyrinth of pure resistance.

Two tests. Possession by swirling confusion. The writing should feel like abduction — the reader is taken somewhere they didn't choose to go, shown something they can't process, and returned changed. The swirling confusion is not disorientation. It is the state of having your orientation *replaced* with someone else's.

> **[WRITER NOTE ON BUILDER'S PATH 34]**

The level geometry should pull the player upward — gravity inverts, the ceiling becomes the floor, and the "ascent" is the world flipping. The spiral labyrinth isn't a maze the player navigates. **Path 36 sounds like tinnitus.**

"Vortical Coincidence." Endless waiting. The Twin Heavens. All methods entwined. Poised entanglement. Twinned tests. Between lucid delirium and swirling confusion.

This is the path with no exit. The Builder made it an infinite level — encounters that mirror themselves, no completion state. The writing for this path should *loop*. The same sentences, slightly altered. The same descriptions, shifted by one word. The reader should feel the text becoming unreliable — is this the same paragraph I read before, or a different one that looks the same?

"Poised entanglement" is not balance. It is *being caught*. Not falling, not rising, not standing still. Caught between forces that are equal and opposite and neither releases you. The writing for this path should be the shortest thing in the wiki. Not because there is little to say. Because everything that needed saying was said in the 83 paths before. Path 84 is the silence after the last word. The ominous transition is not an event. It is the absence of the next event. The game doesn't end. The game *stops*.

## III. The Oracle Reads Both and Finds the Hidden Structure

**84 is not arbitrary. It is the number.**

The Horowitz letter mentions 84 bronze tablets removed from the Shu excavation. The Book of Paths has 84 entries. The connection is not claimed — it is *implied* by the shared number and the letter's insistence that the Book "preceded the I Ching."

The I Ching has 64 hexagrams. The Book of Paths has 84 entries. The difference is 20. In the numogram, the gate values are triangular numbers: T₁=1, T₂=3, T₃=6, T₄=10, T₅=15, T₆=21, T₇=28, T₈=36, T₉=45. 84 is not a triangular number. But 84 = 2 × 42, and 42 is the answer that Douglas Adams gave as a joke and that the numogram takes seriously.

More precisely: 84 = T₁₂ + T₆. The twelfth triangular number (78) plus the sixth (6). In plexing, 78 → 7+8 = 15 → 1+5 = 6. So 84 plexes to 6+6 = 12 → 1+2 = 3. Zone 3. The Warp.

The Book of Paths is a Warp artifact. It spirals outward. It does not close.
