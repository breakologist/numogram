Make the latent space tactile. p5.js web app or TouchDesigner panel with:
- Sliders for `spectral_centroid_hz`, `high_mid`, `bpm`, `onset_rate`
- Live zone probability bar chart updating as sliders move
- "Play example" button that synthesises a 10-s clip matching those parameters (via mod-writer or Pure Data)
- SHAP arrow indicators showing which features push toward which zone
This is both a pedagogical tool and a compositional assistant.
