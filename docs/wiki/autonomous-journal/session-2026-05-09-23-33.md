## Phase 3: Reflect

### Key Findings

1. **Cross-current bridge works end-to-end.** The Roguelike → Numogram → Audio pipeline is operational. Tree dungeon topology maps naturally to musical form: DFS traversal = theme and variations; room depth = section length; zone assignment = harmonic character.

2. **Topology drives distribution, not bias.** The zone distribution skew is a generative property of the tree shape, fundamentally different from the classifier bias documented earlier today. This distinction matters: "why does this dungeon produce mostly zone 3?" is a different question from "why does the classifier label everything zone 2?"

3. **Spectrogram differentiation confirms zone signatures are real.** The vision model identified zone-specific structures independent of knowing the zone assignments. Zone 6's "staircase" pattern and Zone 4's "void gap" are emergent properties, not artifacts of the mapping algorithm.

4. **Limitation — shallow zone spread.** Only zones 1-6 appeared (max depth 6). For full 1-9 coverage: use 15+ rooms, deeper accretion, or non-depth-dependent zone assignment (e.g., cycle zones sequentially through traversal order).

5. **Room size and corridor topology are underutilized.** Room dimensions don't affect musical parameters. Corridors between rooms aren't sonified — they're gaps between sections rather than transitional effects.
