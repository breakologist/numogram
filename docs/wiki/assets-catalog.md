## Usage notes

- All SVGs are vector-based → scale without quality loss
- Source editing: modify original in `~/diagram/` then copy here (or symlink)
- For web export: images will be inlined or copied to `numogram/docs/assets/`
- See also: [[numogram-visualization]] for generation pipeline

## Source directories

- `~/diagram/` — master authoring workspace (ad-hoc SVGs)
- `~/numogram/visualizer/` — auto-generated visualizer output (v6/v7)
- `~/.hermes/obsidian/hermetic/raw/` — scanned source diagrams (PNG/JPG)
