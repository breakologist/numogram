## Overview

Game design principles as applied to the numogram cult-garden and roguelike simulation systems.

## Key concepts
- **Zone as room**: each decimal digit maps to a unique room type
- **Current as corridor**: connections between zones form navigable paths
- **Gate as shortcut**: palindromic/rotational symmetry points enable teleportation
- **Hyperstition threshold systems**: corruption scaling, ability costs, demon aggression at 50/70/85% hyp
