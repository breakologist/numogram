# TC ≡ 2² — The Time-Circuit Axiom

The CCRU holds that the Time-Circuit cluster (TC-set) corresponds
numerically to `2² = 4` inside the 6-cycle of powers of 2.

## Formal Statement

The circular Powers of 2 diagram traces

```
1 → 2 → 4 → 8 → 7 → 5 → 1
T0   T1   T²   T3   T4   T5   T6
```

The centroid node is power `2² = 4`. The TC-set, which collects the
four micro-triangle modes (1/2/4, 5/7/8) traces a second 6-cycle anchored
at the same node — TC ≡ 4.
