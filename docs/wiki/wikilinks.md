## Overview

Conventions for internal wikilinks in the Obsidian numogram vault.

## Syntax
- Basic: `[[page]]`
- With alias: `[[page|display text]]`
- External file: `[[file.ext]]`

## Link hygiene
- Prefer relative links between wiki pages
- Create missing pages as stubs when first linked
- Avoid linking to non-existent skill files; use skill name directly in text
