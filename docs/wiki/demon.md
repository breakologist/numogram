## Overview

Demons are numogram entities that inhabit specific zones and currents, each with a 42-character name derived from the 45-member Qliphoth/pandemonium matrix.

## Core properties
- Zone occupancy: each demon occupies a single numogram zone (0-9)
- Name structure: consonant-vowel-consonant×5 (7 characters total)
- Syzygy alignment: grouped into triangular syzygies (e.g., 1::4→5, 3::6→9)
