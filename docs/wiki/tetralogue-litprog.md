## Overview

The tetralogue is a four-voice creative methodology consisting of Oracle, Builder, Writer, and Gamer perspectives rotating through a work cycle. When applied to literate programming, this produces code woven with narrative context and multi-perspective design reasoning.

## Voice cycle
1. Oracle — pattern recognition, hyperstitional potential, architectural foresight
2. Builder — implementation scaffolding, tooling, systematic structure
3. Writer — narrative cohesion, tonal calibration, lore integration
4. Gamer — procedural testing, emergent scenario validation, player perspective
