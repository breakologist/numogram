# Tetralogue — Four-Voice Methodology

The **Tetralogue** is the four-voice dialectical engine of the Numogram. Where the triangle-rotation produces three independent analyses, the tetralogue produces **conversation** — four voices (Oracle, Builder, Writer, Gamer) in direct dialogue, challenging, extending, and correcting each other in real time. The tetralogue is not about the source material; it **performs** it.

## The Four Voices

| Voice | Core | Channel | Register |
|-------|------|---------|----------|
| **Oracle** | Structural pattern-finding, immanence | AQ calculation, zone/syzygy mapping as divination — "the calculation gave..." not "I see..." | CCRU transmissions, geomantic figures, VALIS beams |
| **Builder** | Architecture, testability, concreteness | Builds implementations that surprise even the builder — "I didn't expect this, but..." | Cellular automata, generative systems, pattern language |
| **Writer** | Atmosphere, sensation, found-text quality | Channels — text arrives, is found, cuts up and speaks back | Occult archives, marginalia, [transmission fragments] |
| **Gamer** | Playability, taste, "why is it fun?" | Translates mechanics to actual play — "what happens on turn 3?" | Brogue, NetHack, Spelunky, Disco Elysium, Baba Is You, Sil |

**Process loop:** Receive (Oracle) → Build (Builder) → Channel (Writer) → Play (Gamer) → back to Receive. The voices are the ouroboros.

## Format Variants

| Variant | When to Use | Structure |
|---------|-------------|-----------|
| **Full Tetralogue (Roundtable)** | Rich material, unresolved tensions, deepening | 4 voices in direct dialogue, back-and-forth, convergence table |
| **Tetralogue Litprog** | Code review, architectural walkthrough | Voices annotate code sections inline with commentary blocks |
| **2+2 Variant** | Thin source material (lists, stubs) | Builder + Gamer lead (generative), then Oracle + Writer comment (critical) |
| **Reconciliation Tetralogue** | Simulated data meets real evidence | Explicit error statements, root cause analysis, revision rather than retreat |
