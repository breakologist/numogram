# Zone Classification Bias Study

## Findings
- All test tracks, regardless of compositional parameters, classified to **Zone 6** (AQ ~50.90)

- Used `mod-writer` CLI with `--render`
- Classified with `predict_audio()` from `mod_writer.classifier`
