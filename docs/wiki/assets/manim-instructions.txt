# Manim Setup Instructions

1. Install Manim:
   pip install manim

2. Render the animation:
   manim -pql /path/to/barker-spiral-working-manim.py

   Options:
   - -p: preview (opens automatically)
   - -q: quiet mode
   - -l: low quality (faster)
   - -h: high quality (slower, better for final)

Note: Manim requires LaTeX to render text. If you don't have LaTeX installed, you'll need to install a TeX distribution:
   - Ubuntu/Debian: sudo apt-get install texlive-full
   - macOS: brew install --cask mactex
   - Windows: Install MiKTeX from miktex.org

Alternative: Use the HTML gallery above to view the visualizations without installing Manim.
