# The Book of Paths

> *The Letter into The Book — Aamodt’s ceremonial framing of the AQ corpus as a living grimoire*

The **Book of Paths** is the canonical 47-entry Alphanumeric Qabbala dictionary treated as a **divinatory compendium** — each phrase is a path through the numogram, each numeric value a gate. It distills the 27 canonical CCRU phrases plus 20 extended entries from later hyperstitional mining (Grok rotor, CC by-nc-sa archives).

## How to read a path

1. **Phrase** → compute AQ value (digital root via numogram-calculator)  
2. **Value** → identifies zone (0–9), current (Time/Warp/Plex), and gate  
3. **Entry** → interpret as oracular response or trigger condition  
4. **Cross-reference** → syzygy partners, demon correspondences, gate maps

- **One-phrase readings:** pick a path at random, meditate on its zone/gate

- **Current traversal:** sequence paths by current (Time-Circuit, Warp, Plex) to trace a journey  
- **Gate activation:** paths whose value equals a gate number (Gt-xx) can "open" that gate
