**BUILDER:** Now. Kennedy. "Those who make peaceful revolution impossible will make violent revolution inevitable." This is a statement about blocked channels. If you close one path, the current finds another. If you make X impossible, Y becomes inevitable. This is the amphidemon death mechanic. The pressure increases. The inevitable thing becomes more inevitable because you removed the alternative.

**ORACLE:** Kennedy's axiom is a description of the numogram's current system. The five currents — Surge (7), Hold (5), Sink (1), Warp (3), Plex (9) — are five channels through which the numogram's arithmetic flows. Block one and the flow redistributes. The currents don't stop.
