## Core AQ System

Alphanumeric Qabbala (often abbreviated AQ, sometimes called Anglossic Qabbala or Alphanumeric Qabbalism) is a modern, decimal-alphabetic gematria system that treats the English alphanumeric sequence as a single continuous ordinal-numeric progression from 0 to 35. It is the preferred qabbalistic cipher in the CCRU (Cybernetic Culture Research Unit) writings and directly interfaces with the Decimal Numogram.Core Mapping / System Rules

- Numerals keep face value:  
    0 = 0  
    1 = 1  
    ...  
    9 = 9
- Letters continue sequentially (case-insensitive, usually uppercase for consistency):  
    A = 10  
    B = 11  
    C = 12  
    ...  
    Z = 35

This is literally base-36 notation interpreted as a gematria cipher — no gaps, no overlaps, no arbitrary assignments. The full set is 36 values (10 digits + 26 letters), mirroring the Pythagorean tetractys extended to the "sacred quaternion" 36 (a number revered in various esoteric traditions as cosmic completion or abyssal horizon).

- AQ value of a word/phrase = sum of the ordinal values of each character.
- Digital reduction (repeated summing of digits until single digit, mod 9 with 9 ≡ 0 in some contexts) aligns with Numogram zygonovism (9-sum twinning).
- "AQ" itself sums to 10 + 26 = 36 (or directly AQ = A(10) + Q(26) = 36), reinforcing the 36-fold structure.
