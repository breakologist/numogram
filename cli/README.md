# CLI Tools

## oracle.py
Complete divination pipeline. Supports:
- `--seed N` — numeric seed
- `--text "phrase"` — AQ-derived seed
- `--hardware` — local entropy
- `--random` / `--blockchain` / `--earthquake` — external APIs
- `--iching` — hexagram casting
- `--taixuan` — T'ai Hsuan two-tetragram
- `--synx` — dual-cipher mode
- `--voice` — generate audio via formant synthesis
- `--json` — machine-readable output
